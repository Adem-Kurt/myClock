# MyClock

A simple desktop clock application built with [LuaRT](https://luart.org/).

## Features

- **Stopwatch:** Measure elapsed time.
- **Timer:** Countdown from a specified time.
- **Pomodoro:** A pomodoro timer to help with productivity.

## Running the Application

To run the application, you will need LuaRT installed. Then, execute the following command in the project root:

```bash
wluart main.lua
```
